# Taller - Tarjetas con LocalStorage

Aplicación web que permite crear tarjetas personalizadas con imagen, nombre y descripción, guardando la información en el LocalStorage del navegador.

---

## Tecnologías usadas

- HTML5
- CSS3
- JavaScript (Vanilla)
- LocalStorage (API del navegador)

---

## Estructura del proyecto

```
taller-tarjetas/
├── index.html
├── app.js
└── style.css
```

---

## Funcionalidades

- Agregar tarjetas con nombre, descripción e imagen (URL)
- Persistencia de datos con LocalStorage
- Eliminar tarjetas individualmente
- Limpiar todas las tarjetas

---

## Conceptos aplicados

- Manipulación del DOM con `createElement` y `setAttribute`
- Eventos con `addEventListener`
- Persistencia con `localStorage.getItem` y `localStorage.setItem`
- Renderizado dinámico de elementos

---

## Cómo usar

1. Abrir `index.html` en el navegador
2. Llenar el formulario con nombre, descripción y URL de imagen
3. Hacer clic en **AGREGAR INFORMACIÓN**
4. La tarjeta aparece en pantalla y se guarda automáticamente
